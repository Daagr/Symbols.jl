# Symbols

### What?

Have you ever wanted to write code like:

- ℜ(τ^𝑖)
- ℱ⁻¹(ℱ(v)) ≈ v
- ⊤⊕⊥
- √n::ℝ = √ℂ(n)
- sin(15°)

Well, I have.

### Notes

Bitwise operations work on booleans too.

You can write `?Symbols` or `@doc Symbols` in the REPL to read this.
