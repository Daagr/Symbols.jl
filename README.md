# Symbols

### What?

Have you ever wanted to write code like:

- F = 𝐺⋅m₁⋅m₂/r^2
- ℜ(τ^𝑖)
- xₙ = 2xₙ₋₁

Well, now you can do some of that.

### Notes

Bitwise operations work on booleans.
